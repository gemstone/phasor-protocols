      Gemstone.PhasorProtocols Namespace

      The Gemstone.PhasorProtocols namespace organizes all Gemstone library functionality related to phasor-protocols. This includes fundamental classes and interfaces used for parsing and generating code specific phasor protocols. The root phasor-protocols namespace also includes common phasor-protocols classes, e.g., MultiProtocolFrameParser.
